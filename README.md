# Dashboard SM Vinhos

Projeto de análise de mercado para segmentação internacional no setor dos vinhos.

## Objetivo
Identificar oportunidades de mercado internacional para exportação de vinhos, analisando indicadores de:

- dimensão de mercado
- crescimento
- preço médio
- potencial de exportação

## Dataset
O dataset utilizado encontra-se no ficheiro:

Mercado Internacional.csv

## Possíveis análises
- ranking de mercados internacionais
- análise de potencial de exportação
- segmentação de mercados premium vs volume
- identificação de mercados emergentes

## Ferramentas
- Excel / CSV
- Data visualization
- Market analysis
